# Hermes Monitoring Plugin

A monitoring plugin for the Hermes Agent ecosystem.

The plugin allows Hermes to:
- Monitor websites and services
- Track uptime and latency
- Manage monitors using AI or the dashboard



## Features

- Website monitoring
- Proxy monitoring
- Application grouping
- Historical ping data
- Hermes tool integration
- Web dashboard



## Available Tools

- `add_website_monitor`
- `list_monitors`
- `remove_monitor`



## Installation

Install it from the "PLUGINS" tab in your Hermes Agent dashboard or place the plugin inside your Hermes plugins directory:

```bash
~/.hermes/plugins/monitoring
```
